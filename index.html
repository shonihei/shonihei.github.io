Antzer

Traditionally, students at my school resorted to trial-and-error when it came down to selecting classes and sections that do not collide with eachother. This often led to frustration as students were forced to either give up their much needed rest to attend an "8am" class or give up their extra-curricular commitments in order to attend class.

This web application is a solution to this issue. Students can list out all the classes they wish to take, blacklist certain hours for rest or for extracurricular activities and the software does the rest!

Pretty Automaton

Python implementation of Elementary Cellular Automaton that allows the user to set custom colors and rule number to generate a pretty image.

For simplicity, two states (on and off) are often represented using two easy- to-dintinguish colors but this implementation allows the user to list several closely related colors to represent a state from which the script chooses a color at random.

BUScraper

This is a Node.js webscraping module made to extract class/section information. It utilizes the popular async package to make HTTP GET calls asynchronously. The school website is structured such that the query result of some class is paginated if all the sections in a class does not fit in a single page. So I used asynchronous programming to simultaneously parse and request additional data.

The module is publicly available on npm.js/buscraper and you can check out my github repository to see how I implemented the module and recommend any fixes.
