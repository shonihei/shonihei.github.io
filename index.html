Class Optimizer

Traditionally, students at my school resorted to trial-and-error when it came down to selecting classes and sections that do not collide with eachother. This often led to frustration as students were forced to either give up their much needed rest to attend an "8am" class or give up their extra-curricular commitments in order to attend class.

This software I wrote in C# is a solution to this issue. Students can list out all the classes they wish to take, blacklist certain hours for rest or for extracurricular activities and the software does the rest!

Pretty Automaton

Python implementation of Elementary Cellular Automaton that allows the user to set custom colors and rule number to generate a pretty image.

For simplicity, two states (on and off) are often represented using two easy- to-dintinguish colors but this implementation allows the user to list several closely related colors to represent a state from which the script chooses a color at random.
